htmlwidgets for R

HTML widgets work just like R plots except they produce interactive web visualations. A line or two of R code is all it takes to produce a D3 graphic or Leaflet map. HTML widgets can be used at the R console for data analysis, shared using RPubs or Dropbox, and even embedded in R Markdown reports and Shiny web applications. Here are a few of the widgets currently available:
Leaflet

Geo-spatial mapping
dygraphs

Time series charting
networkD3

Graph data visualization
DataTables

Tabular data display
threejs

3D scatterplots and globes

Bringing the best of JavaScript data visualization to R

  • Use JavaScript visualization libraries at the R console just like plots.
  • Embed widgets in R Markdown documents and Shiny web applications.
  • Develop new widgets using a framework that seamlessly bridges R and JavaScript.
